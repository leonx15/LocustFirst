To generate certificatem (on Zscaler example) file please follow steps below:

1) Go to any of browsers (ex. chrome)
2) Go to settings > Privacy and Security > Security
3) On this tab search for "Manage Certificates"
4) I the opened window go to "Trusted Root Certification Authorities"
5) Search for "Zscaler Root CA", click on it.
6) Below tab select "Export..."
7) In file format chose step select: "Base-64 encoded X.509 (.CER)"
8) Select place where you want to save certificate.
9) Go to this place and change name and extension of file to "certific.pem"
10) Place this modified file in cert folder in this project.
